# Multi-Skill Container Terminal Planning Assistant

## Overview

Multi-Skill Container Terminal Planning Assistant is an AI-powered decision support prototype designed to assist container terminal planners in analysing vessel operations, resource allocation, yard planning considerations, and operational risks.

The application demonstrates how modern AI agents and skill-based architectures can support planning activities in complex maritime and logistics environments by generating operational recommendations and planning insights from user-provided terminal and vessel information.

This project was developed using Google AI Studio during Google's 5-Day AI Agents Intensive Course.

---

## Key Features

### Vessel Planning Skill

* Analyse vessel workload
* Estimate operation duration
* Evaluate operational complexity

### Quay Crane Allocation Skill

* Recommend crane assignments
* Identify resource shortages
* Support operational balancing

### Yard Planning Skill

* Assess yard congestion
* Suggest yard allocation strategies
* Highlight operational constraints

### Risk Assessment Skill

* Identify operational risks
* Detect potential bottlenecks
* Support proactive planning

### Decision Support Skill

* Generate planning recommendations
* Produce executive summaries
* Provide operational insights for planners

---

## Architecture

The application follows a skill-oriented AI architecture:

```text
Container Terminal Planning Assistant
│
├── Vessel Planning Skill
├── Quay Crane Allocation Skill
├── Yard Planning Skill
├── Risk Assessment Skill
└── Decision Support Skill
```

This design reflects modern Agent Engineering principles where complex decision-support systems are composed of specialised and reusable skills.
